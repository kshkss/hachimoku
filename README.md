# hachimoku
This is a web application that manages your personal finances by tracking how money moves between your different accounts like as double-entry bookkeeping model.

## Features

- **Web Application:** A household budget app accessible from any modern web browser.
- **Single Binary Deployment:** The application is compiled into a single binary, making it easy for users to deploy on their own.
- **Double-Entry Concept:** Manages finances by tracking money movement between accounts, based on a simplified double-entry bookkeeping model.
- **Collaboration:** Each ledger has a single owner, but you can invite other users to collaborate.
